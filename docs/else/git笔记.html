Git笔记

git笔记

  • 初始化本地仓库

    git init

  • 加入暂存区

    git add ,

  • 提交commit

    git commit -m "commit info"

  • 关联远程仓库

    git remote add origin 仓库地址

  • 将远程端版本合并到本地版本中

    git pull origin main

  • 提交远程仓库

    git push remote <remote> <branch>
git push -u origin main

  • 删除远程端分支

    git push <remote> :<branch> (since Git v1.5.0)
git push <remote> --delete <branch> (since Git v1.7.0)

  • 将修改作为当前分支的草稿保存

    git stash

  • 查看草稿列表

    git stash list

  • 删除草稿

    git stash drop stash@{0}

  • 读取草稿

    git stash apply stash@{0}

  • 克隆远程仓库

    git clone 仓库地址

  • 下载远程端版本，但不合并到HEAD中

    git fetch <remote>

  • 显示与上次版本的差异

    git diff

  • 列出所有的分支

    git branch

  • 列出所有的远端分支

    git branch -r

  • 基于当前分支创建新分支

    git brance <new-branch>

  • 基于远程分支创建新的可追溯的分支

    git branch --track <new-branch> <remote-branch>

  • 删除本地分支

    git branch -d <branch>

  • 强制删除本地分支，将会丢失未合并的修改

    git branch -D <branch>
